# Hospital_Emergency_Room_Dashboard
This Excel dashboard provides a dynamic and visual overview of key performance indicators (KPIs) for monitoring and analyzing emergency room operations. Designed for healthcare analysts, hospital administrators, and data enthusiasts, it helps identify bottlenecks, optimize resource allocation, and improve patient experience.

Features

      Daily Patient Volume Tracking Visualize trends in ER visits using area sparklines to detect peak days and seasonal patterns.
      Average Wait Time Analysis Monitor fluctuations in patient wait times to identify delays and improve service efficiency.
      Patient Satisfaction Monitoring Track satisfaction scores to assess service quality and correlate dips with operational challenges.
      Interactive Visuals Includes slicers, conditional formatting, and sparklines for intuitive data exploration.
      Clean Layout & Usability Designed with clarity and simplicity for quick insights and decision-making.
Tech Stack

    Microsoft Excel
    Area Sparklines
    Pivot Tables
    Conditional Formatting
    Basic Formulas (AVERAGE, COUNTIF, etc.)
